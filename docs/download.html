Downloading and installing CAFE

Download

CAFE's files can be downloaded in two different ways:

  1. Download the latest release tarball here, put the tarball in a folder of your choice, and from that folder extract the files with command (replacing "some_version" with the version you downloaded):

    2. $ tar -zxvf CAFE-some_version.tar.gz

  2. If you have git installed, you can clone CAFE's repository by moving to a directory of your choice and entering the command:

    3. $ git clone https://github.com/hahnlab/CAFE.git

Installation

CAFE comes with a precompiled executable file (/path/to/CAFE/release/cafe) that should work on Linux machines. If you prefer to compile from source yourself, move to the CAFE folder, and run:

$ make

Then just copy the binary file to somewhere on your path, such as /usr/local/bin/. Alternatively, add /path/to/CAFE/release/ to your $PATH variable (in your .bashrc or .bash_profile).
